# 3D圣诞树 🎄

一个精美的3D粒子圣诞树，具有旋转动画、发光效果和丰富的装饰。


## 文件结构

```
Christmas tree/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # JavaScript逻辑
├── images/             # 图片文件夹（保留但未使用）
│   ├── crown/          # 树冠图片文件夹
│   ├── trunk/          # 树干图片文件夹
│   ├── leaf/           # 叶子图片文件夹
│   ├── decoration/     # 装饰物图片文件夹
│   │   └── pig.png     # 示例装饰图片
│   └── README.md       # 图片说明文档
└── README.md           # 使用说明
```

## 技术实现

### 核心组件
- **Particle类**：基础粒子类，处理位置、旋转和动画
- **StarParticle类**：星星粒子，具有闪烁效果
- **TopStarParticle类**：顶部大星星，两个垂直相交的五角星
- **GiftParticle类**：礼物盒粒子，带有彩色发光效果
- **OrnamentParticle类**：装饰球粒子，具有闪烁和发光效果
- **ChristmasTree类**：主控制器，管理所有粒子的创建和动画

### 视觉效果
- **树冠**：6层锥形结构，每层粒子数量递减（40→35→30→25→20→15）
- **树干**：均匀分布的圆柱形结构，避免柱状条聚集
- **发光系统**：每个装饰元素都有对应颜色的动态发光效果
- **动画系统**：浮动、闪烁、旋转等多种动画效果

### 设计特点
- **层次分明**：树冠、树干、装饰、星星层次清晰
- **色彩丰富**：多种颜色的礼物盒和装饰球
- **动态效果**：所有元素都有独特的动画和发光效果
- **3D效果**：真正的3D空间分布，旋转时呈现立体感

## 技术栈

- **HTML5**：语义化标记
- **CSS3**：3D变换、动画、渐变
- **JavaScript ES6+**：类、箭头函数、模板字符串
- **SVG**：矢量图形绘制装饰元素

## 自定义选项

代码中提供了多个可调节的参数：
- `particleDensity`：粒子密度
- `starsEnabled`：是否启用星星效果
- 各种尺寸、颜色、动画参数
